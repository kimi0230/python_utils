# Get Captcha Text
Get text from captcha image

## mac
``` shell
brew install tesseract
pip3 install -r requirements.txt
```

## 語言包
https://github.com/tesseract-ocr/tessdata_best 下載並將檔案放到 `/usr/local/Cellar/tesseract/4.1.3/share/tessdata`
